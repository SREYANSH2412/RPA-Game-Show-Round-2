Gal Gadot is conducting research for academic purposes and needs to extract text from PDF documents to incorporate it into her work. Design a workflow to extract the text from each page of the pdf files and display it in a message box with the required details.

Dependances: UiPath.PDF.Activities

To Install the PDF activity pack. On your UiPath Studio go to Manage Packagess, Select Official and type 
"UiPath.PDF.Activities" install the pdf package and you are good to go.

Activities required
-Assign
-For each
-Get pdf page count
-Do while
-Read pdf Text
-Message Box
